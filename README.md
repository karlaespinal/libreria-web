# 📚 Sistema de Librería Web

Este es un sistema web de gestión de libros y autores desarrollado en PHP y MySQL.

## 🚀 Funcionalidades

- 📖 Lista de libros
- ✍️ Lista de autores
- 📩 Formulario de contacto
- 💾 Conexión a base de datos MySQL
